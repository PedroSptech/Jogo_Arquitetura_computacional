# Jogo da Memória — Arquitetura de Computacional

Jogo da memória feito com HTML, CSS e JavaScript para estudar os conteúdos da matéria de Arquitetura de Computacional. O objetivo é encontrar os pares de perguntas e respostas virando as cartas.

---

## Como jogar

1. Abra o arquivo `index.html` no navegador
2. Clique em uma carta para virá-la — ela vai mostrar uma pergunta ou uma resposta
3. Clique em uma segunda carta tentando achar o par correspondente
4. Se as duas cartas formarem um par (pergunta + resposta do mesmo assunto), elas ficam viradas com a borda verde
5. Se não formarem par, as duas viram de volta após um segundo
6. Continue até encontrar todos os 9 pares
7. Ao terminar aparece a tela de vitória com o número de tentativas e o tempo

---

## Conteúdos abordados

As cartas cobrem os seguintes temas:

- O que é a CPU
- O que é a ULA
- O que são os registradores e onde se localizam
- Tipos de memória (RAM, ROM, EPROM, Flash, Memória de massa)
- O que é o DMA e como funciona
- O que é o Chip Select (CS)
- O que é o Address Bus e o Data Bus
- Características dos processadores Intel I5 e I7
- Diferença entre Dual Core e Quad Core
